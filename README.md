## Install

下载发布包：

[dist/hanli-codex-pet.zip](dist/hanli-codex-pet.zip)

解压后，把 `hanli` 文件夹放到 Codex Desktop 的宠物目录：

```text
~/.codex/pets/
```

最终结构应为：

```text
~/.codex/pets/hanli/pet.json
~/.codex/pets/hanli/spritesheet.webp
```

然后重启 Codex Desktop，在宠物列表里选择“韩立”。
